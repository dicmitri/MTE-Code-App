# Data

This directory serves as the "Source of Truth" for all content in the application. It contains JSON files for the MedTech Code text, decision tree logic, and quiz questions.

After editing these JSON files, run `npm run validate:data` from the project root. See [Project Checks](../../PROJECT_CHECKS.md) for plain-language instructions and help interpreting the output.

For more details on the data structure, see the [Project Map](../../AGENTS.md#project-map).
